% Description: this is the front-end file accompanying the back-end thesis.cls file, providing user control of back-end options 
%       & parameters through simple front-end control sequences; i.e. this is the PUBLIC interface to the PRIVATE properties of
%       the .cls file. This is intended to limit direct user interactions with thesis.cls as much as possible while providing 
%       user control over fundamental options/parameters that may need to be changed to satisfy Formatting Guideline
%       changes in the future. 
% Usage: there are 2 main types of user controls...explicit value controls & parameterless Boolean change controls. Value controls
%       are used by simply replacing the existing values with the new desired value. Some value controls may have multiple values,
%       which can ALL be changed if desired. Boolean change controls are used by simply uncommenting the control sequence whose 
%       name describes the desired CHANGE ACTION. Boolean change controls primarily come in \set<AorB> toggle pairs OR individual 
%       \make OR \exclude control sequences,
